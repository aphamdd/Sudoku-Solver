How to run the Simulated Annealing Sudoku Solver:

Simply run the program and it'll attempt to solve Board 2 10 times

To change which board to solve, rename the other numbered puzzles to "puzzle" and
change the original puzzle back to its numbered form. (puzzle2 -> puzzle) (puzzle -> puzzle2)
